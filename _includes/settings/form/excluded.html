excluded: Boolean

Indicate fields which won't be validated.

By default, the plugin will not validate the following kind of fields:

  • disabled
  • hidden
  • invisible

The setting consists of jQuery filters. Accept 3 formats:

Format Description
String

Filters are separated by a comma. For example:

{% highlight javascript %} ':disabled, :hidden, :not(:visible)' {% endhighlight %}
Array of strings

Each element is a filter. For example:

{% highlight javascript %} [':disabled', ':hidden', ':not(:visible)'] {% endhighlight %}
Array of strings and callback functions

The callback function has format as below:

{% highlight javascript %} function($field, validator) { // $field is jQuery object representing the field element // validator is the BootstrapValidator instance // return true or false; } {% endhighlight %}

For example:

{% highlight javascript %} [':disabled', ':hidden', function($field, validator) { // Do not validate invisible element return !$field.is(':visible'); }]{% endhighlight %}

It is also possible to excluded particular field.

The excluded option is usually used when we need to validate the field generated by other UI plugin. For an usage, you can take a look at compatibility examples.
