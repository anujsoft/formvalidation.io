--- layout: post title: Upgrading to v0.5.0 date: 2014-07-14 15:00:00 +0700 author: nghuuphuoc ---

From v0.5.0, we remove the submitHandler option. In v0.5.0, use the success.form.bv event instead.

In v0.4.5 and earlier versions:

{% highlight javascript %} $(form).bootstrapValidator({ submitHandler: function(form, validator, submitButton) { ... } }); {% endhighlight %}

In v0.5.0:

{% highlight javascript %} $(form) .bootstrapValidator({ // Removing submitHandler option }) .on('success.form.bv', function(e) { // Prevent form submission e.preventDefault(); var $form = $(e.target), validator = $form.data('bootstrapValidator'), submitButton = validator.getSubmitButton(); // Do whatever you want here ... }); {% endhighlight %}
